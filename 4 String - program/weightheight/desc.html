Weight and Height (Simplified)

Rian Adam Rajagede
Batas Run-time: 1 detik / test-case
Batas Memori: 32 MB

DESKRIPSI SOAL

Pak Blangkon me-replace 'eight' menjadi '8' pada laporan akhirnya. Dan ternyata dia melakukan kesalahan besar, Kesalahan ini cukup fatal karena dia tanpa sengaja mengubah string 'weight' menjadi 'w8' dan sejenisnya. Bantu Pak Blangkon mengembalikannya.

PETUNJUK MASUKAN

Sebuah bilangan bulat N, menunjukkan akan ada N string yang harus diperbaiki pada N baris setelahnya. String hanya terdiri dari karakter alfabet nonkapital dan angka dengan panjang maksimal 20 karakter.

PETUNJUK KELUARAN

Perbaiki string dengan me-replace '8' dengan 'eight', tapi lakukan ini JIKA HANYA pada string tersebut mengandung minimal satu karakter alfabet. Outputkan masing-masing string dengan berbeda baris.

CONTOH MASUKAN

5
lightw8
wow
89123
8
h8h8h7

CONTOH KELUARAN

lightweight
wow
89123
8
heightheighth7
