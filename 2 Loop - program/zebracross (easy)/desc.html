Zebra Cross (Easy Version)

Rian Adam Rajagede
Batas Run-time: 1 detik / test-case
Batas Memori: 32 MB

DESKRIPSI SOAL

Diberikan input N dan M. Gambarlah zebra cross dengan ukuran NxN.

PETUNJUK MASUKAN

Sebuah bilangan positif N (1 ≤ N ≤ 10) dan sebuah karakter M. Karakter M hanya "H" atau "V".

PETUNJUK KELUARAN

Jika M adalah "H" maka tampilkan zebra cross berbentuk horizontal. Jika "V" tampilkan zebracross berbentuk vertikal. Perhatikan contoh.

5 H
